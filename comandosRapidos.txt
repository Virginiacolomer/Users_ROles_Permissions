Para ingresar en la base con uno de los usuarios cargados y comprobar los permisos asociados:

# CON POSTMAN:

URL: http://localhost:3002/auth/login
Método: POST
Headers:
      Content-Type: application/json

Body (JSON):
      {
      "email": "colomervirginia63@gmail.com",
      "password": "Virginia123"
      }


Lo que nos devuelve el accessToken y refreshToken
Debemos guardar el accessToken para poder hacer peticiones que necesitan autenticacion (Todas menos el registro, inicio de sesion y refreshToken)

Luego para realizar la peticion:

Headers:
      Authorization: Bearer <accessToken>
      Content-Type: application/json

Si el usuario tiene el permiso lo crea, sino, nos lanzara un error 401 o 403 con el mensaje definido en auth.middleware.ts: 
                                'You do not have permission to access this resource'
